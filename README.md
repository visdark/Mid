## Git更换用户要比Github难很多

### git config
基本命令，查询git语句。从列表中选择我们需要操作的命令。

###git config user.email  或者  cat ~/.gitconfig | head -3 
查看用户邮件信息，我们发现，这个邮件和我们所接入的项目账户名不符合，那么我们就需要更改邮箱以及密码的设置。

###git config --global user.email "email"
修改用户名邮件信息。修改名称同上一致。cat ~/.gitconfig | head -3
